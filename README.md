# EMMADUNAMIX-MARCH-2026-ANALYIS
This project analyzes fleet-related expenses for March, providing insights into total spending, frequently purchased items, and driver-level purchase behavior. The analysis identifies cost patterns, repeated purchases, and key expense drivers to support better financial and operational decision-making.

## 🎯 Objectives
- Calculate total expenditure for the month
- Identify top purchased items and their associated costs
- Track purchases by drivers and truck numbers
- Detect repeated purchases across drivers and trucks
- Provide actionable insights for cost optimization

## 🧰 Tools & Technologies
- Python
- Pandas
- Jupyter Notebook
